Kunagi is a software project management tool based on the agile framework Scrum. It is designed for distributed teams of pragmatic ploeple, which need tight collaboration to build a complex product the agile way. kunagi is free.

Features

The Product Backlog is optimized for the Product Owner. Stories are edited inline and priorized by Drag'n'Drop. Based on adjustable velocity, a sprint projection visualizes volume of future releases.

The integrated Wiki provides space for custom documents. Pages can by downloaded as PDF. All description fields (Stories, Tasks, Impediments,...) are based on the Wiki-Syntax which is similar to the MediaWiki-Syntax.
